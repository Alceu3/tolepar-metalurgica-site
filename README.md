# Site Tolepar Metalúrgica

Site institucional estático em HTML, CSS e JavaScript para apresentação da Tolepar Metalúrgica.

## Conteúdo da página
- Hero com chamada principal da empresa
- Seção sobre a Tolepar
- Seção de serviços
- Galeria de trabalhos (blocos prontos para trocar por fotos reais)
- Área de contato com dados da empresa

## Arquivos
- index.html: estrutura da página
- styles.css: identidade visual e responsividade
- script.js: menu mobile e ano automático no rodapé

## Como executar
1. Abra a pasta no VS Code.
2. Abra o arquivo index.html no navegador.
3. Se preferir, use Live Server para atualização automática.

## Observação
Não foi possível extrair automaticamente os trabalhos do Facebook porque a página está com acesso restrito para coleta pública.
Substitua os blocos da seção Trabalhos recentes pelas fotos e descrições reais da Metalúrgica Tolepar.
